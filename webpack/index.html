谁说前端代码不能模块化？

index.js

const foo = require('./foo.js')

foo.bar()

foo.js

module.exports = {
  bar: function () {
    //
  }
}

用 webpack 打包模块。

webpack.config.js

module.exports = {
  entry: './index.js',
  output: {
    path: 'bundle.js'
  }
}

page.html

<html>
  <head>
    ...
  </head>
  <body>
    ...
    <script src="/bundle.js"></script>
  </body>
</html>

很简单。 即刻起步！
